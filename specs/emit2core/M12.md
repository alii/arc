## CRITICAL ARCHITECTURAL DECISION — State is INVISIBLE at the IR level in M12

**Verified against emit_core.gleam:1490-1492, :1530-1535, :3554-3555.** Today `CallHost("js",…)`, `MakeClosure`, `CallClosure` are all state-NEUTRAL under `Threaded` — they do NOT thread `InstanceState`. **M9 changes emit_core so that under `Threaded` + `js_runtime`:**
- `CallHost("js", op, args)` → prepends live `St`, calls `rt_js:t_<op>(St, args…)`, unpacks `{V, St'}`, yields `V`, threads `St'`
- `MakeClosure(f, caps, n)` where `f` is state-reaching → the generated fun body is `apply 'f'/(m+n+1)(St, caps…, args…)`; the fun's arity becomes `n+1`
- `CallClosure(callee, args)` → applies `callee(St, args…)`, unpacks `{V, St'}`, yields `V`, threads `St'`

**Consequence for M12:** the emitter emits IR AS IF state does not exist. `CallHost("js","cell_get",[h])` is written with ONE arg. `CallClosure(f,[this,args])` is written with TWO args. `MakeClosure("f$3",caps,2)` uses arity=2. emit_core (M9) transparently rewrites arity/unpacking. **Every IR function M12 emits has params `[This: TTerm, Args: TTerm]` and result `[TTerm]`.** The BEAM-level `f(St,This,Args)→{V,St'}` shape is M9's output, not M12's.

## The M11 `Build` interface M12 consumes (contract, defined in M11)

```gleam
// M11: arc/compiler/emit_2core/anf.gleam
pub type Build(a) = fn(Emitter2, fn(Emitter2, a) -> ir.Expr) -> ir.Expr
// CPS over Emitter2; the tail continuation receives the final Emitter2 + result.
// `run(b, e)` at the top level supplies `fn(e, v) { ir.Values([v]) }`.

pub fn pure(v: a) -> Build(a)
pub fn then(b: Build(a), f: fn(a) -> Build(b)) -> Build(b)      // monadic bind
pub fn map(b: Build(a), f: fn(a) -> b) -> Build(b)
pub fn seq(bs: List(Build(a))) -> Build(List(a))                // sequence, preserves order

// Let-bind an ir.Expr yielding 1 TTerm; returns fresh Var referencing it.
pub fn bind(rhs: ir.Expr) -> Build(ir.Value)
// Let-bind an ir.Expr yielding N values; returns N fresh Vars.
pub fn bind_n(rhs: ir.Expr, n: Int) -> Build(List(ir.Value))

// CallHost("js", op, args) → bound Var. `host` for stateful ops (M9 threads St);
// `host_pure` for ops resolve_js marks pure (no St prepend — e.g. "type_of","truthy","strict_eq").
pub fn host(op: String, args: List(ir.Value)) -> Build(ir.Value)
pub fn host_pure(op: String, args: List(ir.Value)) -> Build(ir.Value)

// If(cond_i32, [TTerm], then_b, else_b) as an expression; both arms are Build(ir.Value).
pub fn bind_if(cond: ir.Value, t: Build(ir.Value), f: Build(ir.Value)) -> Build(ir.Value)

// Block(label,[TTerm], body) where body may `Break(label,[v])` early. Returns the block result.
pub fn bind_block(body: fn(label: String) -> Build(ir.Value)) -> Build(ir.Value)

// Build a cons list [v0,v1,…] from List(ir.Value) via nested MakeCons + ConstAtom("nil") terminator? No — empty list is CallHost("js","empty_list",[]) OR ir.TermOp(MakeCons,…) chains ending in a bound `[]`. Use:
pub fn cons_list(vs: List(ir.Value)) -> Build(ir.Value)  // right-folds MakeCons over `vs`, terminator = `bind(CallHost("js","empty_list",[]))` — actually the IR has no nil literal; use `host_pure("empty_list", [])` (already in resolve_js:3652).

// The guarded fast/slow arithmetic (HANDOFF §5).
pub fn guarded_binop(fast: ir.NumTermOp, slow_op: String, l: ir.Value, r: ir.Value) -> Build(ir.Value)
// = bind(If(TermTest(IsNumber,l)&&TermTest(IsNumber,r), NumTerm(fast,l,r), CallHost("js",slow_op,[l,r])))
// where `&&` = `Let([a],TermTest(IsNumber,l), If(Var(a),TermTest(IsNumber,r),Values([ConstI32(0)])))`

// Compare variant returning i32 → boxed to atom true/false.
pub fn guarded_cmp(fast: ir.NumTermOp, slow_op: String, l: ir.Value, r: ir.Value) -> Build(ir.Value)
// = fast path yields i32 → wrap via `bind(If(i32, Values([ConstAtom("true")]), Values([ConstAtom("false")])))`
```

## Sentinel/atom vocabulary (defined by M3 rt_js_val, used here as `ir.ConstAtom`)

Per `twocore_rt_js_ffi.erl` header + rt_js.gleam:31:
- `undefined`, `null`, `true`, `false` — the four atom values
- `js_nan`, `js_inf`, `js_neg_inf` — number sentinels (ast.gleam has no NaN literal; only via `0/0` etc. at runtime, and `InfiniteNumber` for parsed `Infinity`)
- BigInt encoding (M3 owns): `{js_bigint, Int}` tagged 2-tuple (distinguishes `1n` from `1` for `typeof`)

Each row: `ast.Expression` variant → IR shape via M11 combinators. `⇒` = `then`; `↦` = yields.

### Identifier (`ast.Identifier(_, name)`)

Source: emit.gleam:1800-1848, scope.gleam:2083, :317-342.

```
match name:
  "undefined" → pure(ConstAtom("undefined"))       // emit.gleam:4282
  "#" <> rest → fail(EarlySyntaxError("Unexpected private name #" <> rest))  // emit.gleam:4286
  _ → emit_identifier(name)
```

`emit_identifier(name)`:
```
let res = scope.lookup(e.tree, e.current_scope, name)  // via M10 accessor
case res:
  Plain(Local(slot, boxed: False, kind, _)) →
    // TDZ check for let/const/class before init — M10 tracks `initialized: Set(Int)`
    // per emit.gleam:1841. If kind ∈ {LetBinding,ConstBinding,ClassBinding} and
    // slot ∉ e.initialized → emit runtime TDZ check:
    //   bind_if(host_pure("is_tdz",[Var(slot_var(slot))]),
    //           host("throw_reference_error",[ConstBinary(name)]), pure(Var(slot_var(slot))))
    // Simpler + matches arc: TDZ uses a SENTINEL value (M3 defines atom `js_tdz`);
    // initialized let/const overwrite it. Read: Var(slot_var(slot)) always;
    // TDZ is caught lazily when the value is USED (rt_js ops all reject `js_tdz`).
    // DECISION: eager check matching arc → host_pure("tdz_check",[Var(slot_var(slot)), ConstBinary(name)])
    //   which returns v unchanged or throws. Only when M10 says slot may be uninit.
    pure(Var(state.slot_var(slot)))  // wrapped in tdz_check when needed
  Plain(Local(slot, boxed: True, ..)) →
    host("cell_get", [Var(state.slot_var(slot))])
  Plain(Global(name)) →
    host("global_get", [ConstBinary(<<name:utf8>>)])
  Plain(EvalEnv(name)) →
    host("eval_env_get", [ConstBinary(<<name:utf8>>)])   // M4 op; direct-eval only
  WithChain(crossed, fallback) →
    // crossed: List(SlotRef) — each is a cell holding the with-object.
    // Emit nested If chain probing each with-obj via has_prop; on hit, get_prop;
    // on miss-all, recurse into `fallback` as Plain(fallback).
    emit_with_chain_get(crossed, name, fallback)
```

`emit_with_chain_get` (private):
```
fold_right(crossed, emit_direct_get(fallback, name), fn(acc, SlotRef(slot, boxed)) {
  use wobj <- then(read_slot(slot, boxed))  // Var or cell_get
  use has  <- then(host("has_prop", [wobj, ConstBinary(name)]))
  use hi32 <- then(host_pure("truthy", [has]))  // i32
  bind_if(hi32, host("get_prop", [wobj, ConstBinary(name)]), acc)
})
```

### `TemplateLiteral(_, TemplateParts(head, tail))`

Source: emit.gleam:5059-5078. §13.2.8.5 uses ToString (string hint), NOT `+`'s ToPrimitive.
```
use acc0 <- then(pure(ConstBinary(<<head:utf8>>)))
list.fold(tail, pure(acc0), fn(acc_b, #(e, quasi)) {
  use acc <- then(acc_b)
  use v   <- then(expr(e))
  use s   <- then(host("to_string", [v]))          // M3 to_string, string-hint
  use a1  <- then(host_pure("string_concat", [acc, s]))   // NOT js `+`; direct binary concat (M3)
  host_pure("string_concat", [a1, ConstBinary(<<quasi:utf8>>)])

```
`string_concat` is a NEW M3 pure op (2 binaries → binary; no coercion). Avoids `add`'s ToPrimitive.

### `BinaryExpression(_, op, left, right)`

Special-case FIRST (emit.gleam:4294): `In` with LHS = `Identifier("#" <> name, ..)` → private-brand check:
```
use r <- then(expr(right))
use k <- then(emit_identifier("#" <> name))  // resolves to the minted private-key local (M16 sets up)
host("private_in", [r, k])                    // M7 op
```

Generic: `use l <- then(expr(left)); use r <- then(expr(right)); binop(op, l, r)` where `binop` is the operator table below.

### `LogicalExpression(_, op, left, right)`

Source: emit.gleam:4312-4319, HANDOFF:170. Short-circuit: RHS evaluated only if needed. **RHS must be inside the If arm** (side-effects).
```
use l <- then(expr(left))
case op:
  LogicalAnd →
    use t <- then(host_pure("truthy", [l]))  // → i32
    bind_if(t, expr(right), pure(l))
  LogicalOr →
    use t <- then(host_pure("truthy", [l]))
    bind_if(t, pure(l), expr(right))
  NullishCoalescing →
    use t <- then(host_pure("is_nullish", [l]))  // M3 op: undefined|null → 1, else 0
    bind_if(t, expr(right), pure(l))
```
`truthy` is PURE (rt_js.gleam:3638 — no proto walk; typeof-class check only). `is_nullish` NEW pure M3 op.

### `UpdateExpression(_, op, prefix, target)`

Source: emit.gleam:4392-4486. First `unwrap_parens(target)`. Then:

### `MemberExpression` / `OptionalMemberExpression` / `OptionalCallExpression`

**Optional-chain routing** (emit.gleam:4834, ast_util.gleam:432): if `chain_has_optional(expr)` → `emit_chain_root(expr)`. Else plain path.

**Plain `MemberExpression(_, obj, prop)`**:
- `obj = SuperExpression` → `emit_super_get(prop)` (M16 helper — reads RefHomeObject proto + RefThis receiver, `host("get_super_prop",[home_proto,key,this])`).
- else → `expr(obj) ⇒ fn(o) emit_member_get(o, prop)`:
  - `Dot(name)` where `name = "#" <> _` → `host("private_get",[o, <resolve private key local>])` (M7)
  - `Dot(name)` → `bind(anf.object_key_lit(KeyIdentifier(name, _))) ⇒ fn(k) host("get_prop", [o, k])`
  - `Bracket(k)` → `expr(k) ⇒ fn(kv) host("to_property_key",[kv]) ⇒ fn(pk) host("get_prop", [o, pk])`

**`emit_chain_root(expr)`** (source emit.gleam:4107-4174, translated from label-jump to structured `Block`):
```
bind_block(fn(exit) {
  emit_chain(expr, exit)
})
// where emit_chain recursively descends links; at each `?.` boundary:
//   use base <- then(emit_chain(inner, exit))
//   use n    <- then(host_pure("is_nullish", [base]))
//   use _    <- then(bind(If(n, Break(exit,[ConstAtom("undefined")]), Values([]))))  // 0-result If as guard
//   … continue with base as receiver
// Method-call shape (a?.b.m(x)) preserves receiver: the chain carries an
// Option(ir.Value) receiver alongside the current value.
```

`emit_chain` signature (private): `fn(ast.Expression, exit: String) -> Build(#(ir.Value, Option(ir.Value)))` — `(value, receiver_if_method_shape)`. Mirrors emit.gleam:4133-4174 depth-1/depth-2 distinction but with structured Break instead of stack-cleanup labels.

### `CallExpression(_, callee, args)`

Source: emit.gleam:4641-4798. Evaluation order §13.3.6.2: callee ref → args → call.

1. `chain_has_optional(callee)` → `emit_chain_root(CallExpression(...))` (chain compiler handles the call).
2. `callee = SuperExpression` → **super()** (emit.gleam:4641-4673): `emit_lexical(RefActiveFunc) ⇒ host("get_prototype_of",[af]) ⇒ ctor; emit_lexical(RefNewTarget) ⇒ nt; emit_args(args) ⇒ argsL; host("construct",[ctor,argsL,nt]) ⇒ inst; emit_lexical_put(RefThis, inst); <field-init call if M10.field_init>; pure(inst)`.
3. `callee = MemberExpression(_, SuperExpression, prop)` → super.m(args): `emit_super_get(prop) ⇒ f; emit_lexical(RefThis) ⇒ t; emit_args(args) ⇒ aL; emit_call(f, t, aL)`.
4. `callee = MemberExpression(_, obj, prop)` (method call — `this`=obj): `expr(obj) ⇒ o; emit_member_get(o,prop) ⇒ f; emit_args(args) ⇒ aL; emit_call(f, o, aL)`.
5. `callee = Identifier("eval")` (no spread) → **direct eval** (emit.gleam:4740): FIXED DESIGN says full JS semantics; direct eval requires runtime compilation. `emit_identifier("eval") ⇒ f; emit_args(args) ⇒ aL; host("direct_eval",[f, aL, <scope-snapshot>])` where scope-snapshot is a cons_list of the `with_stack` cell handles + boxed-locals map — M4 op that (a) checks `f === intrinsic eval` at runtime, (b) if so recompiles+runs against snapshot, (c) else regular call. *(Design-level: direct-eval-in-compiled-code is HANDOFF §6 territory; this routes to rt_js which invokes arc's own compile pipeline against the live JsStore. M4/M6 own the impl.)*
6. `unwrap_parens(callee) = Identifier(name)` inside `with` (M10.with_stack ≠ []) → `emit_with_callee(name) ⇒ #(f, recv); emit_args(args) ⇒ aL; emit_call(f, recv, aL)` where `emit_with_callee` returns receiver=with-obj on hit, undefined on miss (emit.gleam:4778-4797, :1810).
7. else → `expr(callee) ⇒ f; emit_args(args) ⇒ aL; emit_call(f, ConstAtom("undefined"), aL)`.

**`emit_args(args: List(Expression)) -> Build(ir.Value)`** — builds the args cons-list. If any `SpreadElement`:
```
// Build a JS array via host ops, then host("array_to_list",[arr]) → cons list.
use arr <- then(host("array_new", []))
fold(args, arr, fn(a, arg) case arg {
  SpreadElement(_, e) → expr(e) ⇒ fn(v) host("array_spread_into",[a, v])  // M4: iterates v, pushes
  _                   → expr(arg) ⇒ fn(v) host("array_push",[a, v])
}) ⇒ fn(arr) host("array_to_list",[arr])
```
No spread → `seq(list.map(args, expr)) ⇒ cons_list`.

**`emit_call(f, this, argsL) -> Build(ir.Value)`** — the uniform call site (HANDOFF:172):
```
use is_fun <- then(bind(TermTest(IsFun, f)))
bind_if(is_fun,
  bind(CallClosure(f, [this, argsL])),           // M9 threads St → f(St,this,argsL)→{V,St'}
  host("call_not_fun", [f, this, argsL]))        // M5: handles native fns wrapped as tuples,
                                                 // Proxy [[Call]], or throws TypeError "not callable"
```

### `ObjectExpression(_, properties)`

### `LValue` protocol (private type + 3 fns)

Source: emit.gleam:1918-1929 (VarRef), :4452-4481 (member get2/put). Evaluates base/key ONCE.
```gleam

  LvIdent(name: String, direct: scope.Direct, with_base: Option(ir.Value))
    // with_base = Some(wobj) if a with-chain probe HIT; None → static fallback
  LvMember(obj: ir.Value, key: ir.Value, private: Bool)
  LvSuper(home_proto: ir.Value, key: ir.Value, this: ir.Value)
}
```
- `emit_lvalue(Identifier(n))` → resolve; if WithChain, probe now (before RHS runs — §13.15.2 step 1a) via `host("with_resolve",[wobj,ConstBinary(n)])` returning `{hit_bool, base}`; store base or fallback Direct.
- `emit_lvalue(MemberExpression(_,SuperExpression,prop))` → `LvSuper(emit_lexical(RefHomeObject)⇒host("get_prototype_of"), emit_key_from_prop(prop), emit_lexical(RefThis))`.
- `emit_lvalue(MemberExpression(_,o,prop))` → `expr(o) ⇒ ov; emit_key_from_prop(prop) ⇒ kv; LvMember(ov,kv, is_private(prop))`.
- `lvalue_get`: `LvIdent`→read direct/with_base; `LvMember`→`host(private?"private_get":"get_prop",[obj,key])`; `LvSuper`→`host("get_super_prop",[hp,key,this])`.
- `lvalue_put(lv,v)`: `LvIdent`→`emit_direct_put`/`host("set_prop",[with_base,key,v])`; check `kind=ConstBinding`→`host("throw_type_error",["Assignment to constant"])`; `LvMember`→`host(private?"private_set":"set_prop",[obj,key,v])`; `LvSuper`→`host("set_super_prop",[hp,key,this,v])`. Returns `pure(v)`.

`emit_identifier_put(name, v)` = `emit_lvalue(Identifier(name)) ⇒ lvalue_put(_, v)`.

## Invariants M12 upholds

1. **Every `emit(e, named)` yields exactly one `ir.Value : TTerm`.** Never `TI32`/`TF64` — comparison i32 results are immediately boxed to `true`/`false` atoms; NumTerm arithmetic results are already `TTerm`.
2. **Evaluation order matches ECMAScript** (§13): left→right for binops; callee before args; obj before key before value in `o[k]=v`; base/key evaluated exactly ONCE for compound assign (LValue protocol).
3. **No State variable appears in emitted IR.** M12 relies on M9's state-threading for `CallHost("js",…)` / `CallClosure` / `MakeClosure`.
4. **NamedEvaluation (§8.4)** flows via the `named: Option(String)` parameter, dropped at `ParenthesizedExpression`, applied only to anonymous fn/arrow/class.
5. **Short-circuit ops evaluate RHS inside the `If` arm**, never eagerly.
6. **No `ast.Expression` variant falls through** — the `case` is exhaustive (36 variants); `SuperExpression`/`SpreadElement` standalone → `Error(EarlySyntaxError(_))` (unreachable per parser, but fail closed).
7. **`chain_has_optional` gates every Member/Call read** so `a?.b.c` short-circuits the whole chain (emit.gleam:4834).

## Gotchas / integration risks

- **emit_core.gleam:1490-1492 blocks state-threaded closures TODAY.** M9 MUST land the `MakeClosure`/`CallClosure` state-threading change before M12's `emit_call` compiles correctly. Until then any JS→JS call under `Threaded` will lose store mutations. **Hard dependency: M12 blocks on M9.**
- **`ConstF64(bits)` stores raw IEEE bits as Int** (ir.gleam:476). Gleam has no float→bits builtin; needs `<<f:float-big-64>>` → `bit_array` → int conversion (8-byte big-endian). M11 or M12 provides `ieee754_bits/1`.
- **`ast.LiteralNumber` has NO NaN variant** (ast.gleam:367-370) — only `FiniteNumber|InfiniteNumber`. `NaN` is an `Identifier("NaN")` → global lookup. So NumberLiteral never emits `js_nan`.
- **rt_js today is pdict-based** (rt_js.gleam:33, twocore_rt_js_ffi.erl "process dictionary"). M1 replaces this with threaded `{js_cell, Int}` store. M12's `host(…)` calls are name-stable across that change.
- **`strict_eq` must be pure** but `eq` is stateful (ToPrimitive). M9's op classification must reflect this or `1=={valueOf(){…}}` won't observe the store.
- **Bitops on objects need ToPrimitive** → all bitops are `host` (stateful), not `host_pure`, despite the temptation.
- **`truthy` returns i32** (resolve_js:3638 comment "i32 1|0") — M12 uses it directly as `If.cond` (which wants i32 per ir.gleam:683).
